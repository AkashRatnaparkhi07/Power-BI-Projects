HR Analytics Dashboard | Power BI Project
This project is an end-to-end HR Analytics Dashboard built using Power BI Desktop, designed to help HR managers and business leaders monitor workforce health, understand employee attrition, and make data-driven decisions about talent management.
Project Overview
The dashboard analyzes key HR metrics including employee attrition rate, headcount, department-wise distribution, salary bands, job satisfaction, and performance ratings. The goal was to identify patterns behind employee turnover and provide actionable insights to reduce attrition.
Key Features
Interactive KPI cards display total employees, attrition count, attrition rate, and average age. Slicers allow filtering by department, gender, education field, and job role. Charts include bar charts for department-wise attrition, pie charts for gender distribution, and line graphs for attrition trends over time. A salary analysis page breaks down compensation by job level and experience.
DAX Measures Used
Custom DAX measures were created for attrition rate percentage, average monthly salary, employee satisfaction index, and tenure-based segmentation. Time intelligence functions track month-over-month changes in headcount.
Data Preparation
Raw HR data was cleaned and transformed using Power Query — removing duplicates, handling null values, creating conditional columns for age groups and salary brackets, and building a proper data model with relationships between fact and dimension tables.
Tools Used
Power BI Desktop, DAX, Power Query, Microsoft Excel
What I Learned
This project strengthened my understanding of data modelling, DAX measures, and designing dashboards that are both visually clean and functionally rich. It showed me how analytics can directly support workforce planning and retention strategy decisions.
